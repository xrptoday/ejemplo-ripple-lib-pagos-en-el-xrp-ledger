ripple-lib: pagos en el XRP Ledger
=========

Código de ejemplo para ilustrar cómo se pueden procesar pagos en el XRP Ledger con la ayuda de ripple-lib.

## Installation

  `git clone && npm install`

## Run

  `node app`
